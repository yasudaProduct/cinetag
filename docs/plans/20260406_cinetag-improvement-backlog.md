# cinetag 改善バックログ（提案の整理）

作成日: 2026-04-06

このドキュメントは、プロダクト改善の候補を整理したものです。

---

## 優先度が高い候補

### マイページを「管理画面」として完成させる

- プロフィール上の統計（例: 作成タグ数）の正確な表示
- `GET /api/v1/me/tags` の実装（仕様上は未実装）とマイページ連携
- 自分のタグの一覧・非公開タグの扱い・整理しやすい導線

### タグ編集体験の強化

- タグ内映画の**メモ編集**・**表示順の変更**（仕様上 `PATCH .../movies/:tagMovieId` は未実装）
- **カバー画像**の設定（UI/API の一本化）
- 複数追加・整理のしやすさ

### フォロー起点のホームフィード

- フォロー中タグの更新、フォロー中ユーザーの新規タグなどを一覧化するフィード
- API 仕様の「今後の拡張候補」（`feed/me` 等）に沿った設計の検討

### 映画詳細からの導線強化

- 映画ページから**新規タグ作成**・**既存タグへ追加**へのショートカット
- 「この映画を含むタグ」からの関連タグ・発見導線の拡充

### 通報・モデレーション

- タグ・ユーザーの通報、スパム対策、運用者向けの非公開化・BAN など
- API 仕様の管理系拡張候補と整合

---

## 中期で効きそうな候補

### 視聴ステータス

- 「観た」「観たい」などを映画またはタグ内エントリに紐づける（要スキーマ・UX 設計）

### 共同編集の質

- 追加理由・一言コメント・並びの意味付けなど、文脈が伝わるタグ体験

### おすすめ・発見

- TMDB メタデータとタグの共起などに基づくタグ・映画の提案

### 共有・SEO

- タグ・プロフィールの OGP、シェア文言、構造化データの継続改善
